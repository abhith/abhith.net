## Monorepo

In case of monorepo which contains many Pipelines, we keep the build Pipelines yml files under,

repo > build\azure-devops\{project-name}\azure-pipelines.yml

## Naming Convention

### Build Pipelines

#### Pipeline Name

Project Name-CI

### Release Pipelines

#### Pipeline Name

Project Name-CD

#### Release name format

$(Build.DefinitionName)-Release-$(rev:r)

#### Archive file to create

E:\Backups\{webiste-name}-$(Build.BuildId)-$(System.JobId).zip

#### Stages

- Dev
- Test
- Production
- Sandbox
